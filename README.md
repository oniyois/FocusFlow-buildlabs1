# FocusFlow

A responsive landing page for a study productivity tool built for a Frontend Developer Internship task.

## Features

- Responsive navigation bar
- Hero section
- Features section
- Testimonial slider
- Mobile hamburger menu
- Hover effects and transitions
- Fully responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Responsive Design

The project is optimized for:
- Mobile devices
- Desktop screens
